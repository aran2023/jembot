# 🤖 JEMBOT BLE LED 제어기

이 프로젝트는 **Web Bluetooth API**를 사용하여 웹 브라우저에서 ESP32-S3의 LED를 제어하는 시스템입니다.

## 🚀 주요 기능
* **무선 연결:** 전용 앱 없이 블루투스로 직접 연결
* **간편 제어:** 웹 페이지의 버튼으로 LED ON/OFF 제어
* **HTTPS 배포:** GitHub Pages를 이용한 보안 호스팅

## 🛠 사용 방법
1. ESP32-S3에 `BLE_LED_Main.ino` 코드를 업로드합니다.
2. 이 저장소의 웹사이트 주소에 접속합니다.
3. [장치 연결] 버튼을 눌러 `Jembot_S3_LED`를 선택합니다.
